# bourse
Bourse PyTool / Stockoption python tool with ichimoku

## tti
I use package tti and v0.2.2 needs to be patched
So see and apply from patch directory.
See applypatch.sh script

## Stock option isin
tool uses isin information from a CSV file which is named 'tickers.csv'
Format is ISIN CODE, NAME (without column name) like:
QS0009079318,"AMUNDI LABEL EQUILIBRE SOLIDAIRE ESR - F"
